# Delaunay-Triangulations
Using Delaunay Triangulations for competitive assement


## Overview
The script in this Git geospitally maps locations of stores in Australia

There are three additions to the standard mapping based on two blog posts:
* [circls](http://gis.stackexchange.com/questions/119736/ggmap-create-circle-symbol-where-radius-represents-distance-miles-or-km)
* [Delaunay Triangulations] (https://datascienceplus.com/metro-systems-over-time-part-2/)
* [Voronoi Diagrams] (http://letstalkdata.com/2014/05/creating-voronoi-diagrams-with-ggplot/)

All attemp to depict the competitive environment.
